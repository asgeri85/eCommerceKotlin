# E-Commerce-App

### E-Commerce application with MVVM & Clean Arch - Hilt - Coroutines - Retrofit - Coil - Room DB - Lottie - OkHttp - Navigation.

## Project Features
 - MVVM with Clean Architecture
 - Coroutines
 - Navigation Components
 - Dependency Injection - Dagger-Hilt
 - ROOM Database
 - REST API
 - Retrofit
 - Coil
 - OkHttp 
 - Gson
 - Data Binding
 - View Binding
 - Lottie
